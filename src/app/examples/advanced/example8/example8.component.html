Example 8: Two-way Currency Converter

Update either input to convert between US Dollar ($) and Euro (€) using signals and setter pattern.

Current: {{ dollar() | number:'1.2-2' }} $ = {{ euro() | number:'1.2-2' }} €

Both inputs are two-way bound and update each other using signals.
<input [value]="dollar()" (input)="setDollar($any($event.target).valueAsNumber)" />
<input [value]="euro()" (input)="setEuro($any($event.target).valueAsNumber)" />
