Example 6 - Form Defaults That Depend on Another Field

When a user selects a country, the default currency updates automatically. The user can override the currency, but if the country changes again, the currency resets to the new default.

Country @for (c of countries; track c) { {{c.name}} } Currency @for (cur of availableCurrencies; track cur) { {{cur}} }

Tip: Change the country to see the currency auto-reset. Manually select a currency to override. Changing country again resets to default.
